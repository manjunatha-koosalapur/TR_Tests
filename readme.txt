Steps to use test scripts:
1. Download or clone this repo
2. Install dependencies by running the following command 
   npm install
   npm install cypress <in case change in cypress version>
3. Run tests from the Command Line Interface, by running the following command
   npx cypress run
4. Run tests from the Cypress Test Runner
   npx cypress open
   Select E2E Testing
   Start E2E Testing in Chrome (or any other borwser)
   Select 'europeTours_testSuite.cy.js' from E2E specs list
